ve la siguiente información de la manera más detallada posible: como un desarrollador web experimentado, crea un juego simple y limpio al estilo flappy bird usando html, css y javascript. el juego debe incluir: un pájaro que se mueve verticalmente al hacer clic o pulsar una tecla, tuberías que se desplazan horizontalmente con espacios aleatorios para pasar, detección de colisiones para terminar el juego, y un contador de puntos que aumenta cada vez que el pájaro pasa por una tubería. El código debe ser claro, bien organizado y comentado para fácil lectura y mantenimiento. no uses librerías externas ni frameworks, solo código nativo para navegador web.
